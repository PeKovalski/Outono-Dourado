# Outono Dourado - Site de Produtos Sustentáveis

## 🍂 Sobre o Projeto

Site completo da empresa **Outono Dourado**, especializada em produtos sustentáveis e orgânicos localizada em Araucária, Paraná.

## 📁 Estrutura do Projeto

O site é composto por apenas **3 arquivos essenciais**:

- `index.html` - Estrutura HTML completa
- `style.css` - Estilos CSS com tema "Outono Dourado"
- `script.js` - JavaScript vanilla para funcionalidades

## 🚀 Como Usar

1. **Para rodar localmente:**
   - Simplesmente abra o arquivo `index.html` em qualquer navegador
   - Ou use um servidor local como Live Server (VS Code)

2. **Para hospedar no GitHub Pages:**
   - Faça upload dos 3 arquivos para seu repositório
   - Ative o GitHub Pages nas configurações do repositório
   - O site estará disponível em: `https://seu-usuario.github.io/nome-do-repositorio`

## ✨ Funcionalidades

- ✅ Sistema completo de carrinho de compras
- ✅ Catálogo de produtos com filtros por categoria
- ✅ Seções responsivas: Hero, Sobre, Produtos, Lojas, Contato
- ✅ Interação do mascote Milhito
- ✅ Formulário de contato funcional
- ✅ Navegação suave entre seções
- ✅ Sistema de notificações toast
- ✅ Links para mapas das lojas e WhatsApp
- ✅ Design totalmente responsivo
- ✅ Tema "Outono Dourado" (#DF760B e #F6B61E)
